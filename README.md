# Projet-technique-
Nous allons fabriquer une petite boîte musicale électronique. L'utilisateur choisit un son de batterie, par exemple  kick, snare, hi-hat ou clap, puis active des pas rythmiques sur une grille. La machine lit les pas en boucle selon un  tempo réglable. Chaque fois qu'un pas activé arrive, le Teensy déclenche le sample audio correspondant.
